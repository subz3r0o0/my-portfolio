Xplore Games

A comprehensive game tracker app that personalises the gaming experience for players.

Project Overview

The Xplore Games app is a dedicated game tracker suitable for all gamers. It allows users to explore personalized game recommendations, track progress, stay up-to-date with gaming news, and discover the world of retro gaming.

Development Process

Xplore Games was developed using Xcode and Swift. My process involved wireframing the app’s interface and using the Swift programming language to create multiple views, each functioning to enhance user experience. Key resources such as Apple's documentation, Hacking with Swift, and Stack Overflow articles helped guide the development.

Challenges and Solutions

During development, I faced difficulties embedding a database and used protocols like Hashable and Identifiable to resolve some of the issues. The login system is not fully implemented, but future iterations will include API integration to allow users to sign in via gaming accounts such as Microsoft’s Xbox.
